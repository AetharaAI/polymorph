# Gemini Project Rules

Read machine-readable project state first:
- `project-state/ai/manifest.yaml`
- `project-state/ai/current-state.yaml`
- `project-state/ai/runtime-contracts.yaml`
- `project-state/ai/repo-map.yaml`

Then read human docs only if more context is required.

Never treat narrative docs as the sole source of runtime truth.
