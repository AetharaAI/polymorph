# Claude Project Rules

Before acting, read:
- `project-state/ai/manifest.yaml`
- `project-state/ai/current-state.yaml`
- `project-state/ai/runtime-contracts.yaml`
- `project-state/ai/repo-map.yaml`

Use human docs only when the machine-readable state is insufficient.

When changing behavior:
- update the relevant AI contract/state file
- update the human changelog if the change affects runtime behavior or migration
